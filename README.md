# flutter20251227

這是一個簡單的 Flutter Web 專案，包含 BMI 計算器範例。

## 先決條件
- 安裝 Flutter (stable) 並加入 `PATH`
- 啟用 Flutter Web 支援：`flutter config --enable-web`
- 安裝 Firefox (可選，若要直接用 `-d firefox` 裝置)

## 安裝與執行
1. 在專案目錄執行依賴安裝：
	```bash
	flutter pub get
	```
